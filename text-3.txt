$ git diff
برای مشاهده تغییرات ایجاد شده درون سند ها

$ git log
مشاهده کامیت ها ، نویسنده و تاریخ آن ها

$ git log --oneline
فقط مشاهده کامنت هر کامیت

$ git log -2 --oneline
فقط مشاهده دو کامنت آخر

$ git log -p
مشاهده تغغبراتی که در هر کامیت به وجود اومده
اول تغییرات کامیت آخری رو نشون میده. برای دیدن تغییرات کامیت های قبلی دکمه اسپیس رو بزن

$git log --stat
نمایش تعداد حذفیات و اضافیات به وجود امده در هر کامیت
