git init 
برای ایجاد خودکار مخزن گیت

git status
نمودار شاخه ای کار های انجام شده (working tree)

git add .
تمام تغغیرات در به طور موقت استیج ثبت میشه

git commit -m "message"
برای ثبت تغییرات در ریپازتوری با همون مخزن

git commit -a -m "message"
ثبت هر دو مورد بالایی (البته نه برای اولین بار. اولین بار باید از همون ادد استفاده کرد)
