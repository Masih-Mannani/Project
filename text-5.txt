$ git branch <name>  or  $ git checkout -b <name>
ایجاد یک شاخه یا ورژن دیگر برای پروژه

$ git branch -a
مشاهده تمام شاخه های پروژه

$ git checkout <name>
سوئیچ کردن روی شاخه ی دلخواه

$ git checkout -d <name>
حذف شاخه دلخواه به شرطی که روی آن شاخه نباشیم

$ git merge <name>
مرج کردن فایل دلخواه روی شاخه ای که رویش نشسته ایم

$ git log --graph
مشاهده شاخه هایی که به فایل کنونی مرج یا برنچ کرده ایم

ممکن است موقع مرج کردن ، به خاطر تشابه محتوای شاخه ها ، گیت دچار سردرگمی شود و از ما سوال کند که کدام یک از این محتوا را مرج کند... بعد از انخاب به صورت دستی کافی است آن فایل نهایی را ادد و کامیت کنیم
