$ git checkout -- <file name>
برگرداندن تغییرات در یک فایل مدیفای شده

$ git checkout -- .
برگرداندن تغییرات در همه ی فایل های مدیفای شده

$ git restore --staged < file name or . >
برگرداندن فایل یا فایل هایی که به درون استیج ادد کردیم . در واقع فایل ها از استیج به درخت کاری ما بر میگردند

$ git reset <commit id>
کامیت های بعد از آن بر میگردند (حالا یا حذف میشوند یا دوباره ثبت میشوند)

$ git reset --hard <commit id>
تمام تغییرات بعد از این کامیت به صورت کامل حذف یا ثبت می شوند
